================================================================================================== ================================================================================================ git branch git branch Lista, cria ou exclui branches. ________________________________________________________________________________________________ git branch -d Excluir um branch após mesclar todas as alterações _____________________________________________________________________________________________ git branch -D Excluir um branch independentemente das alterações (forçado) ================================================================================================ git checkout git checkout -- Muda para um branch específico ou restaura arquivos. =============================================================================================== git merge Combina as alterações de uma branch em outra. =============================================================================================== git push --set-upstream origin : Faz o push do branch local para o repositório remoto chamado "origin" e configura esse branch remoto como o upstream, facilitando futuros comandos de push e pull. _______________________________________________________________________________________________ git config --global push.default current: Configura globalmente o Git para empurrar o branch atual para um branch remoto com o mesmo nome por padrão, quando git push é executado sem especificar explicitamente o branch remoto e local. ============================================================================================== git pull --set-upstream origin : Faz pull do branch remoto "origin/" para o branch local , configurando automaticamente o branch remoto como upstream. ___________________________________________________________________________________________________ git pull -u origin : Faz o mesmo que o comando anterior, mas usando a opção abreviada -u para configurar o upstream durante o pull. ============================================================================================= git reflog ============================================================================================== git log Exibe um histórico de commits.
